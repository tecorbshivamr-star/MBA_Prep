## 6.2 Introduction: Meaning of Capital Structure

### Definitions

| Term | Definition |
|---|---|
| **Financial Structure** | **Total** of all financing sources — short-term + long-term debt + equity |
| **Capital Structure** | Only the **long-term** financing mix — LT debt + preference + equity |
| **Capitalisation** | Total long-term funds (LT debt + equity) at book value |
| **Leverage / Gearing** | Degree to which debt is used in the capital structure |

> 📌 **Capital Structure ⊂ Financial Structure** — Capital Structure is a subset.

### The Central Question:

> *"Does the mix of debt and equity (capital structure) affect the total value of the firm and its cost of capital?"*

This is the **central problem of capital structure theory** — different theories give different answers.

---

### Key Concepts: Value of Firm and WACC

```
Value of Firm (V) = Value of Debt (D) + Value of Equity (E)
                  = EBIT(1−t) / WACC   [in perpetuity]

Goal: Find the capital structure that MAXIMISES V
      which is equivalent to MINIMISING WACC
```

---

### EBIT-EPS Analysis (Practical Tool)

**Definition**: Examines the relationship between EBIT and EPS under different financing alternatives to identify the **indifference point** (where EPS is equal for both alternatives).

```
EPS = [EBIT − Interest] × (1 − Tax Rate) / Number of Shares

Indifference Point: Find EBIT* where EPS(Plan A) = EPS(Plan B)
```

#### Example:
> Company needs ₹10,00,000. Two options:
> - **Plan A (Equity)**: Issue 10,000 shares at ₹100 each. Current shares = 20,000. Total = 30,000 shares.
> - **Plan B (Debt)**: Borrow at 10% interest = ₹1,00,000/yr. Shares remain 20,000.
> Tax rate = 30%.

**Indifference Point:**
```
[EBIT − 0] × 0.70 / 30,000 = [EBIT − 1,00,000] × 0.70 / 20,000

0.70 × EBIT / 30,000 = 0.70 × (EBIT − 1,00,000) / 20,000

Multiply both sides by 60,000:
2 × EBIT = 3 × (EBIT − 1,00,000)
2×EBIT = 3×EBIT − 3,00,000
EBIT* = ₹3,00,000

At EBIT > ₹3,00,000 → Debt (Plan B) gives higher EPS
At EBIT < ₹3,00,000 → Equity (Plan A) gives higher EPS
```

## 6.3 Factors Affecting Capital Structure Decisions

| Factor | Explanation |
|---|---|
| **Business Risk** | Higher operating risk → Use less debt (already risky enough) |
| **Tax Position** | Profitable firms benefit more from interest tax shield → Use more debt |
| **Financial Flexibility** | Need to maintain ability to raise funds in emergencies → Keep some debt capacity |
| **Managerial Attitude** | Conservative managers prefer low debt; aggressive prefer high leverage |
| **Lender / Rating Agency Attitude** | Debt covenants and credit ratings constrain debt levels |
| **Sales Stability** | Stable revenues (utilities, FMCG) → Can support more debt |
| **Asset Structure** | Firms with tangible assets → Can pledge as collateral → More debt |
| **Growth Rate** | High-growth firms → Prefer equity (to preserve financial flexibility) |
| **Profitability** | Highly profitable firms generate internal funds → Need less external debt |
| **Control** | Promoters wanting to retain control → Prefer debt over new equity |
| **Market Conditions** | Bullish market → Issue equity; Bearish → Issue debt |
| **Size of Firm** | Large firms → Better access to debt markets at lower rates |
| **Industry Norms** | Firms tend to cluster around industry average leverage ratios |

## 6.4 Theories of Capital Structure

### Overview of All Four Theories:

| Theory | Does Capital Structure Affect V? | Optimal Capital Structure Exists? |
|---|---|---|
| **NI (Net Income)** | ✅ Yes | ✅ Yes — 100% debt |
| **NOI (Net Operating Income)** | ❌ No | ❌ No — irrelevant |
| **Traditional** | ✅ Yes | ✅ Yes — moderate debt |
| **M&M (No Tax)** | ❌ No | ❌ No — irrelevant |
| **M&M (With Tax)** | ✅ Yes | ✅ Yes — 100% debt |
| **Trade-Off Theory** | ✅ Yes | ✅ Yes — balanced |

---

### Theory 1: Net Income (NI) Approach

**Proponent**: David Durand

**Core Assumption**: Both Kd (cost of debt) and Ke (cost of equity) remain **constant** regardless of leverage. Since Kd < Ke, using more debt lowers WACC → Increases firm value.

**Conclusion**: The more debt, the lower the WACC, and the **higher the firm value**.
→ Optimal structure = **100% Debt**

```
As Debt ↑:
  Kd = Constant (cheap debt)
  Ke = Constant (no risk perception change)
  ∴ WACC ↓
  ∴ Firm Value ↑

Maximum V → 100% Debt Capital Structure
```

```
Cost
 │  Ke (constant, higher)
 │──────────────────────
 │
 │          Ke
 │   WACC ↓ as debt ↑
 │──────────────────────────
 │  Kd (constant, lower)
 │──────────────────────────
 └──────────────────────────► Debt/Equity Ratio
```

**Criticism**: Unrealistic — Ke does NOT stay constant as risk increases with leverage.

---

### Theory 2: Net Operating Income (NOI) Approach

**Proponent**: David Durand (alternative view)

**Core Assumption**: WACC is **constant** regardless of leverage. As debt increases, Ke rises exactly enough to offset the benefit of cheaper debt → WACC unchanged → Firm value unchanged.

**Conclusion**: Capital structure is **irrelevant** — no optimal structure.

```
As Debt ↑:
  Kd = Constant
  Ke = Rises (shareholders demand higher return for financial risk)
  WACC = Constant (Ke rise exactly offsets debt benefit)
  ∴ Firm Value = Constant (EBIT/WACC = unchanged)
```

**Criticism**: Ke may not rise perfectly to keep WACC constant; ignores tax shield.

---

### Theory 3: Traditional (Intermediate) Approach

**Proponent**: Ezra Solomon

**Core Assumption**: Initially, debt is cheap and Ke doesn't rise much → WACC falls. But beyond a point, Ke rises sharply and lenders also demand higher Kd → WACC rises. There is a **range of optimal leverage**.

**Conclusion**: There IS an optimal capital structure at moderate debt levels — a **U-shaped WACC curve**.

```
Cost / WACC
 │
 │  Ke (rising steeply at high leverage)
 │                        ╱
 │                       ╱
 │  WACC ╲           ╱
 │         ╲_____╱  ← Optimal Range (minimum WACC)
 │
 │  Kd (rising gradually)
 │
 └──────────────────────────────────► Debt Ratio
              ↑
         Optimal Capital Structure
         (Minimum WACC = Maximum Firm Value)
```

**Criticism**: Does not specify exactly where the optimal point is.

---

### Theory 4: Modigliani–Miller (M&M) Approach — The Most Important ✅

**Proponents**: Franco Modigliani & Merton Miller (1958, 1963)

#### M&M — Without Corporate Tax (1958)

**Assumptions (Perfect Capital Market):**
- No taxes
- No transaction costs
- Individuals can borrow at same rate as firms
- Symmetric information
- No bankruptcy costs
- Firms classified into homogeneous risk classes

**Two Propositions:**

**Proposition I (Firm Value)**:
```
V_L = V_U

Value of a Levered Firm = Value of Unlevered Firm
(Capital structure is IRRELEVANT to firm value)
```

**Proposition II (Cost of Equity)**:
```
Ke_L = Ke_U + (Ke_U − Kd) × (D/E)

Ke rises proportionally with leverage, exactly offsetting the benefit of cheaper debt
∴ WACC remains constant regardless of leverage
```

**Mechanism — Arbitrage:**
If V_L ≠ V_U, rational investors will use **arbitrage** (personal leverage) to eliminate the difference:
- If V_L > V_U: Sell levered firm shares, borrow personally, buy unlevered shares → V_L falls, V_U rises until equal.

---

#### M&M — With Corporate Tax (1963)

**Key Change**: Interest on debt is **tax-deductible** → Creates a **Tax Shield**.

**Proposition I (with Tax)**:
```
V_L = V_U + T × D

Where:
  T   = Corporate tax rate
  D   = Market value of debt
  T×D = Present Value of Tax Shield (perpetual)

∴ V_L > V_U → Use MORE debt to maximise firm value
```

**Proposition II (with Tax)**:
```
Ke_L = Ke_U + (Ke_U − Kd) × (D/E) × (1 − T)

Ke still rises with leverage, but LESS steeply (due to tax shield)
∴ WACC DECREASES as leverage increases
```

**Conclusion (with Tax)**: Optimal structure = **100% Debt** (to maximise tax shield).

---

### M&M — Full Summary Table (Must Memorise!):

| Feature | **M&M Without Tax** | **M&M With Tax** |
|---|---|---|
| **Firm Value (V)** | V_L = V_U (Irrelevant) | V_L = V_U + T×D (Debt adds value) |
| **WACC** | Constant regardless of leverage | Decreases as debt increases |
| **Cost of Equity (Ke)** | Rises with leverage | Rises with leverage (less steeply) |
| **Optimal Structure** | Does not exist | 100% Debt |
| **Tax Shield** | Not considered | PV of Tax Shield = T × D |
| **Year** | 1958 | 1963 |

---

### Trade-Off Theory (Post M&M Extension) ✅

**Concept**: Firms trade off the **tax shield benefits of debt** against the **costs of financial distress** (bankruptcy costs + agency costs).

```
V_L = V_U + PV(Tax Shield) − PV(Financial Distress Costs)

Optimal Capital Structure: Where the marginal tax shield benefit
= Marginal financial distress cost
```

```
Firm Value
     │
     │          ← Optimal leverage (Max V)
     │         ╱╲
     │        ╱  ╲ PV(Financial Distress Costs)
     │       ╱    ╲
V_U  │──────╱      ╲──────
     │     ╱ PV(Tax  ╲
     │    ╱  Shield)  ╲
     └────────────────────────► Debt Ratio
          ↑
    Optimal D/E Ratio
```

**Key Predictions:**
- Firms with **high tax rates** should use **more debt** (bigger tax shield)
- Firms with **tangible assets** → Can sustain more debt (low distress cost)
- **High-growth / intangible-heavy firms** → Less debt (high distress cost)
- Debt ratios **revert to target** (mean reversion) over time

---

### Pecking Order Theory (Myers & Majluf, 1984)

**Concept**: Due to **information asymmetry**, firms prefer financing in a specific **pecking order**:

```
PECKING ORDER OF FINANCING PREFERENCE:

1st → Retained Earnings (Internal Funds)     [No information leakage]
2nd → Debt (External borrowing)              [Less information problem]
3rd → New Equity Issue                       [Most costly signal of overvaluation]
```

**Logic**: Managers know more than investors (information asymmetry). Issuing new equity signals that managers think the stock is **overvalued** → Stock price falls on announcement.

**Key Predictions:**
- **No target D/E ratio** — leverage is just a residual
- **Profitable firms** use **less debt** (more retained earnings available)
- **High-growth firms** issue **more debt** than mature firms (retain flexibility)
- Firms have a **financing hierarchy**, not an optimal structure

---

### Trade-Off Theory vs. Pecking Order Theory:

| Feature | **Trade-Off Theory** | **Pecking Order Theory** |
|---|---|---|
| **Target Leverage** | ✅ Yes — specific D/E target | ❌ No target |
| **Basis** | Tax shield vs. distress cost | Information asymmetry |
| **Profitable Firms** | Use more debt (bigger tax benefit) | Use less debt (internal funds sufficient) |
| **Equity Issuance** | Used when debt is excessive | Last resort (signals overvaluation) |
| **D/E Ratio Pattern** | Reverts to target | Drifts based on financing needs |
| **Empirical Support** | Good for cross-sectional variation | Good for time-series behaviour |

## 6.5 Checklist for Optimal Capital Structure

The **optimal capital structure** minimises WACC and maximises firm value. No single formula — managers use a checklist:

| Checklist Item | Consideration |
|---|---|
| **1. Business Risk** | Lower operating risk → Higher debt capacity |
| **2. Tax Position** | Higher tax rate → Greater benefit from interest tax shield |
| **3. Asset Tangibility** | Physical assets → Better collateral → More debt |
| **4. Growth Opportunities** | High growth → Lower debt (preserve flexibility) |
| **5. Profitability** | High profits → Internal funds available → Less debt needed |
| **6. Firm Size** | Larger firms → Better access to debt markets |
| **7. Market Conditions** | Issue equity in bull markets; debt in bear markets |
| **8. Industry Norms** | Stay close to industry average leverage |
| **9. Lender Requirements** | Maintain minimum interest coverage ratios |
| **10. Control Concerns** | Debt preferred when promoters want to avoid dilution |
| **11. Credit Rating** | Maintain investment-grade rating → Limits debt |
| **12. Flexibility Reserve** | Keep unused debt capacity for emergencies |

> 📌 **Rule of Thumb**: Optimal D/E ratio balances **tax benefits** against **financial distress risk**, customised for the firm's specific characteristics.

## 6.6 Financial Distress and Capital Structure

### Financial Distress

**Definition**: Situation where a firm **struggles to meet its financial obligations** — a stage before formal bankruptcy.

```
Continuum:
Financial Difficulty → Financial Distress → Insolvency → Bankruptcy
```

| Stage | Meaning |
|---|---|
| **Financial Difficulty** | Tight cash flow; delayed payments |
| **Financial Distress** | Unable to service debt; covenant violations |
| **Insolvency** | Liabilities > Assets (balance sheet insolvency) or cannot pay debts (cash flow insolvency) |
| **Bankruptcy** | Formal legal process (IBC 2016 in India; Chapter 11 in USA) |

---

### Costs of Financial Distress

#### A. Direct Bankruptcy Costs:
```
Visible, out-of-pocket costs:
├── Legal fees (lawyers, courts)
├── Accounting and advisory fees
├── Administrative costs of restructuring
└── Asset liquidation costs (fire-sale discounts)
```

#### B. Indirect Bankruptcy Costs (More Significant!):
```
Hidden costs — often larger than direct costs:
├── Lost customers (fear firm won't honour warranties/service)
├── Lost suppliers (unwilling to extend trade credit)
├── Lost employees (talent leaves for stable firms)
├── Forgone investment opportunities (no funds for +NPV projects)
├── Management distraction (time spent on financial crisis, not operations)
└── Fire-sale asset prices (assets sold cheap in distress)
```

---

### Agency Costs of Debt

When firms are in financial distress, **shareholders and managers may act against debtholders' interests**:

| Problem | Explanation |
|---|---|
| **Asset Substitution (Risk Shifting)** | Near-bankruptcy shareholders prefer risky projects: gains go to them, losses borne by debtholders |
| **Underinvestment** | Shareholders reject +NPV projects if most value goes to debtholders |
| **Milking the Property** | Paying excess dividends to shareholders when firm is near distress |
| **Playing for Time** | Delaying bankruptcy filing hoping for a turnaround (at creditors' expense) |

**Debtholders anticipate these and respond with:**
- **Restrictive covenants** (limit dividends, new debt, asset sales)
- **Higher interest rates** (risk premium for agency risk)
- **Collateral requirements**
- **Monitoring costs** (audits, reporting requirements)

---

### PV of Financial Distress Costs:

```
PV(Financial Distress Costs) = Probability of Distress × Cost if Distress Occurs

This rises sharply at high leverage levels

Optimal Capital Structure (Trade-Off):
  V* = V_U + PV(Tax Shield) − PV(Financial Distress Costs)
```

---

### India's Insolvency Framework — IBC 2016

| Feature | Detail |
|---|---|
| **Full Name** | Insolvency and Bankruptcy Code, 2016 |
| **Regulator** | IBBI (Insolvency and Bankruptcy Board of India) |
| **Time Limit** | Resolution must be completed in 180 days (extendable to 330 days) |
| **Initiator** | Financial creditor, operational creditor, or corporate debtor |
| **Process** | CIRP (Corporate Insolvency Resolution Process) |
| **Outcome** | Resolution Plan OR Liquidation |
| **Purpose** | Time-bound, creditor-friendly resolution vs. old BIFR system |

### ⚡ Master Formula Sheet

```
Value of Firm:
  V = D + E = EBIT(1−t) / WACC

EBIT-EPS Indifference Point:
  [EBIT − I_A](1−t) / n_A = [EBIT − I_B](1−t) / n_B

M&M Proposition I (No Tax):
  V_L = V_U

M&M Proposition I (With Tax):
  V_L = V_U + T × D      [T×D = PV of Tax Shield]

M&M Proposition II (No Tax):
  Ke_L = Ke_U + (Ke_U − Kd)(D/E)

M&M Proposition II (With Tax):
  Ke_L = Ke_U + (Ke_U − Kd)(D/E)(1−T)

Trade-Off Theory:
  V_L = V_U + PV(Tax Shield) − PV(Financial Distress Costs)

WACC (General):
  WACC = Ke(E/V) + Kd(1−T)(D/V)

EPS:
  EPS = [EBIT − Interest] × (1 − Tax Rate) / No. of Shares
```
