## 5.2 Introduction

> Management makes decisions every day: Should we accept a low-price order? Should we make a component or buy it? Which product to push when resources are limited?
> **Cost concepts** give the financial logic behind every such decision.

---

## 5.3 Cost Concepts in Decision Making ⭐

| Cost Concept | Definition | Use in Decision |
|-------------|-----------|----------------|
| **Marginal Cost** | Cost of producing ONE extra unit (= Variable Cost) | Core of marginal costing |
| **Opportunity Cost** | Value of next best alternative given up | Make or Buy, resource allocation |
| **Sunk Cost** | Already spent, cannot be recovered | IGNORE in all decisions |
| **Differential Cost** | Difference in cost between two options | Comparing alternatives |
| **Relevant Cost** | Future cost that changes with the decision | Include in analysis |
| **Irrelevant Cost** | Does NOT change with decision (fixed costs, sunk costs) | Exclude from analysis |

> 🧠 **BugZero Rule:** When making any decision, **only count costs that change**. Fixed salary, past spending = irrelevant. New variable costs = relevant!

---

## 5.4 Objectives of a Costing System
1. **Ascertain cost** of product/service accurately
2. **Control costs** by comparing actual vs standard
3. **Aid pricing decisions**
4. **Assist planning** through budgets and forecasts
5. **Help decision making** — Make/Buy, shut down, etc.
6. **Measure efficiency** of departments/processes

## 5.5 Marginal Costing ⭐⭐⭐

> **Marginal Costing** = A costing technique where only **Variable Costs** are charged to products. **Fixed Costs** are treated as **period costs** and written off entirely against profit in the period they occur.

### 📌 Key Principle:
```
MARGINAL COST = VARIABLE COST
(Direct Material + Direct Labour + Direct Expenses + Variable Overhead)

FIXED COST → NOT a product cost → Written off to P&L directly
```

### 📌 Contribution ⭐⭐⭐ (The Heart of Marginal Costing)

> **Contribution** = Amount that each unit **contributes** toward covering Fixed Costs and then generating Profit.

```
Contribution = Sales − Variable Cost
Contribution = Fixed Cost + Profit
Profit       = Contribution − Fixed Cost
```

### 📋 Marginal Costing Income Statement Format

```
₹           ₹
Sales Revenue                        XX,XXX
Less: Variable Cost of Sales        (XX,XXX)
                                    ────────
CONTRIBUTION                         XX,XXX
Less: Fixed Costs (Period Cost)     (XX,XXX)
                                    ────────
NET PROFIT / LOSS                    XX,XXX
```

> 🧠 **BugZero Analogy:**
> BugZero charges ₹1,00,000 per testing contract.
> Variable cost per contract = ₹40,000 (engineer time, tools used)
> **Contribution = ₹60,000** per contract
> Fixed costs (rent, internet, salaries) = ₹3,00,000/month
> Profit = (₹60,000 × No. of contracts) − ₹3,00,000

---

## 5.6 Advantages of Marginal Costing

| Advantage | Explanation |
|-----------|-------------|
| **Simple** | No complex overhead absorption |
| **Profit not affected by stock** | Fixed cost not carried in closing stock |
| **Better for decisions** | Contribution analysis guides pricing, mix decisions |
| **Highlights cost behavior** | Clearly separates fixed and variable |
| **BEP analysis possible** | Essential for CVP analysis |
| **Avoids over/under absorption** | No absorption = no over/under issue |

---

## 5.7 Limitations of Marginal Costing

| Limitation | Explanation |
|-----------|-------------|
| **Separation difficulty** | Hard to split semi-variable costs into fixed and variable |
| **Ignores fixed costs in pricing** | May lead to underpricing long-term |
| **Not accepted by GAAP** | Financial statements require absorption costing |
| **Not suitable for long-term** | Fixed costs matter in the long run |
| **Stock undervalued** | Closing stock doesn't include fixed costs |

---

## 5.8 Marginal Costing vs Absorption Costing ⭐⭐

| Feature | Marginal Costing | Absorption Costing |
|---------|-----------------|-------------------|
| **Fixed Cost treatment** | Period cost → fully to P&L | Product cost → absorbed into units |
| **Stock Valuation** | At variable cost only | At full cost (variable + fixed) |
| **Profit** | Differs based on stock levels | Higher if closing stock > opening stock |
| **Technique** | Contribution approach | Total cost approach |
| **Decision Making** | Better suited | Less suitable |
| **Statutory** | Not accepted | Accepted (Ind AS / GAAP) |

### 🔢 Profit Reconciliation Formula ⭐

```
When Closing Stock > Opening Stock:
  Absorption Profit > Marginal Profit

When Closing Stock < Opening Stock:
  Marginal Profit > Absorption Profit

Difference = Change in Stock × Fixed OH per unit

Marginal Profit + (Closing Stock − Opening Stock) × Fixed OH rate
= Absorption Profit
```

### 📊 Solved Example: Marginal vs Absorption Profit

**Data:** Production = 1,200 units. Sales = 1,000 units.
SP = ₹100/unit. Variable cost = ₹60/unit. Fixed OH = ₹24,000.
Fixed OH rate = ₹24,000 / 1,200 = ₹20/unit.

```
MARGINAL COSTING:
Sales (1,000 × ₹100)             = ₹1,00,000
Less: Variable Cost (1,000 × ₹60) = (₹60,000)
Contribution                      = ₹40,000
Less: Fixed Cost (full amount)    = (₹24,000)
Profit                            = ₹16,000

ABSORPTION COSTING:
Sales (1,000 × ₹100)             = ₹1,00,000
Less: Full Cost (1,000 × ₹80)    = (₹80,000)  [₹60 + ₹20 fixed]
Profit                            = ₹20,000

Difference = ₹20,000 − ₹16,000 = ₹4,000
= Closing Stock (200 units) × Fixed OH rate (₹20) = ₹4,000 ✅
```

## 5.9 Cost-Volume-Profit (CVP) Analysis ⭐⭐⭐

> **CVP Analysis** = Study of the **relationship between Cost, Volume (output), and Profit** to understand how changes in output affect profitability.

### 🔢 P/V Ratio (Profit-Volume Ratio) ⭐⭐⭐

> Also called **Contribution/Sales Ratio (C/S Ratio)**. Shows what % of sales is contribution.

```
P/V Ratio = Contribution / Sales × 100
          = (Sales − Variable Cost) / Sales × 100
          = 1 − (Variable Cost / Sales)
          = Change in Profit / Change in Sales × 100
```

> 🔑 **Higher P/V Ratio = More profitable product**

### 🔢 Relationship Between Key Metrics:
```
Profit   = Sales × P/V Ratio − Fixed Cost
Sales    = (Fixed Cost + Desired Profit) / P/V Ratio
Contribution = Sales × P/V Ratio
```

---

## 5.10 Break-Even Analysis ⭐⭐⭐

> **Break-Even Point (BEP)** = The level of output/sales at which **Total Revenue = Total Cost** → **Zero Profit, Zero Loss**.

### 🔢 BEP Formulas (MEMORIZE ALL ⭐)

```
BEP (in Units) = Fixed Cost
                 ─────────────────────────
                 Contribution per unit (S−V)

BEP (in ₹)    = Fixed Cost
                 ─────────────
                 P/V Ratio

BEP (in ₹)    = BEP (units) × Selling Price per unit
```

### 🔢 Margin of Safety (MOS) ⭐⭐

> **MOS** = The amount by which actual sales **exceed** Break-Even Sales. Shows how much sales can fall before a loss occurs.

```
MOS (₹)  = Actual Sales − BEP Sales

MOS (%)  = (Actual Sales − BEP Sales) / Actual Sales × 100
          = Profit / Total Contribution × 100

MOS (₹)  = Profit / P/V Ratio

Profit   = MOS × P/V Ratio
```

> 🧠 **BugZero Analogy:**
> BugZero's BEP = 5 contracts/month (just covers rent + salaries).
> Currently doing 8 contracts → **MOS = 3 contracts** — that's the safety buffer!
> If orders drop by 3, they break even. If by more than 3, they make a loss.

### 📊 Solved Numerical: BEP + MOS

**Given:** Fixed Cost = ₹60,000. Selling Price = ₹200/unit. Variable Cost = ₹120/unit.
Actual Sales = 1,000 units.

```
Contribution per unit = ₹200 − ₹120 = ₹80
P/V Ratio             = 80/200 × 100 = 40%

BEP (units)   = ₹60,000 / ₹80 = 750 units
BEP (₹)       = ₹60,000 / 0.40 = ₹1,50,000

Actual Sales  = 1,000 × ₹200 = ₹2,00,000
MOS (₹)       = ₹2,00,000 − ₹1,50,000 = ₹50,000
MOS (%)       = ₹50,000 / ₹2,00,000 × 100 = 25%

Actual Profit = MOS × P/V Ratio = ₹50,000 × 40% = ₹20,000
Verification: Contribution (1,000 × ₹80) − FC = ₹80,000 − ₹60,000 = ₹20,000 ✅
```

### 📌 Target Profit Sales

```
Sales to Achieve Target Profit = (Fixed Cost + Target Profit) / P/V Ratio
Units for Target Profit        = (Fixed Cost + Target Profit) / Contribution per unit
```

## 5.11 Decision-Making Problems ⭐⭐⭐

### 📌 Decision 1: Make or Buy

> Should we **manufacture** a component in-house or **buy** it from outside?

**Rule:**
```
If Cost of Making < Purchase Price → MAKE
If Cost of Making > Purchase Price → BUY

Compare: Variable Cost of Making vs Purchase Price
(Ignore fixed costs if they remain the same either way)
```

**Solved Example:**
BugZero needs 1,000 testing dongles. Cost to make in-house:
- Variable cost = ₹45/unit, Fixed cost = ₹20/unit (unavoidable)
- Purchase price from vendor = ₹60/unit

```
Relevant cost of making = Variable cost only = ₹45 (fixed cost is unavoidable)
Purchase price          = ₹60

Since ₹45 < ₹60 → MAKE in-house → Saving = ₹15 × 1,000 = ₹15,000
```

> **But if fixed costs can be SAVED by buying (avoidable fixed costs):**
> Then compare: (Variable + Avoidable Fixed) vs Purchase Price

---

### 📌 Decision 2: Accept or Reject Special Order

> A customer offers a **price lower than normal** for a bulk order. Accept?

**Rule:**
```
Accept if: Selling Price of Special Order > Variable Cost (Marginal Cost)
           i.e., Contribution from order > 0

Condition: Factory has SPARE CAPACITY (no opportunity cost)
If no spare capacity: Also consider contribution lost from regular orders
```

**Solved Example:**
BugZero's normal price = ₹1,00,000/contract. Variable cost = ₹40,000.
A new client offers ₹55,000 for one contract. Spare capacity exists.

```
Contribution from special order = ₹55,000 − ₹40,000 = ₹15,000 > 0
Decision: ACCEPT → adds ₹15,000 to profit even at lower price!
```

> 🧠 **Logic:** Fixed costs are already covered by regular business. Any price above variable cost adds to profit!

---

### 📌 Decision 3: Key Factor / Limiting Factor ⭐⭐

> When a **resource is scarce** (machine hours, labour hours, raw material), which product should be prioritized?

**Rule:**
```
Rank products by: Contribution per unit of Limiting Factor
(NOT by total contribution or P/V ratio alone)

Contribution per unit of Key Factor = Contribution per unit / Units of Key Factor per unit
```

**Solved Example:**
BugZero has only **200 QA engineer hours** available. Two projects:

| Project | Contribution | Hours Required | Contribution per hour |
|---------|-------------|----------------|----------------------|
| Alpha   | ₹60,000      | 40 hours        | ₹1,500/hr            |
| Beta    | ₹90,000      | 80 hours        | ₹1,125/hr            |
| Gamma   | ₹50,000      | 25 hours        | ₹2,000/hr ← Best!   |

```
Ranking: Gamma (₹2,000) > Alpha (₹1,500) > Beta (₹1,125)

Allocation of 200 hours:
  Gamma: 25 hrs → Contribution = ₹50,000 (remaining: 175 hrs)
  Alpha: 40 hrs → Contribution = ₹60,000 (remaining: 135 hrs)
  Beta:  80 hrs → Contribution = ₹90,000 (remaining: 55 hrs)
         (55/80 of Beta) → Contribution = 55/80 × ₹90,000 = ₹61,875

Total Contribution = ₹50,000 + ₹60,000 + ₹61,875 = ₹1,71,875
```

---

### 📌 Decision 4: Shut Down or Continue

> Should a **loss-making department/product** be closed?

**Rule:**
```
Continue if: Contribution > 0
             (Even if making a loss after fixed costs, contribution covers some fixed costs)

Shut Down if: Variable Cost > Revenue (negative contribution)
              OR if savings from shutting > contribution earned

Saving from shut down = Avoidable Fixed Costs only
```

**Solved Example:**
BugZero's "Manual Testing" division:
- Revenue = ₹2,00,000
- Variable Cost = ₹1,50,000 → Contribution = ₹50,000
- Fixed Cost (unavoidable) = ₹80,000 → Net Loss = ₹30,000

```
If shut down: Lose ₹50,000 contribution, save ₹0 (fixed costs unavoidable)
Net loss worsens by ₹50,000 → CONTINUE the division!

Decision: CONTINUE as long as Contribution > 0
```

---

### 📌 Decision 5: Profitable Product Mix

> Which combination of products maximizes **total contribution**?

**Rule:** When NO limiting factor → produce all with positive contribution
When limiting factor exists → rank by contribution per unit of that factor (same as Decision 3)
