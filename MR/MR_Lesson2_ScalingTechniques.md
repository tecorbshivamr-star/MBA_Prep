## 1. 📖 INTRODUCTION

### What is Measurement?
> **Measurement** = Assigning numbers or labels to objects/events according to rules, to represent quantities of attributes.

- In Marketing Research, we measure **attitudes, opinions, perceptions, preferences**
- Example: "How satisfied are you with the product?" → We assign numbers (1 to 5)
- The tool used to measure = **Scale**

### What is Scaling?
> **Scaling** = The process of placing respondents on a continuum with respect to their attitude toward an object.

- Scaling creates a **continuum** (a range) for measurement
- Example: From "Strongly Disagree (1)" to "Strongly Agree (5)"

## 2. 📐 MEASUREMENT SCALES

There are **4 types** of measurement scales — this is foundational knowledge!

> [!TIP]
> **Mnemonic: "NOIR"** (Like the French word for "Black" — mysterious and cool!)
> - **N**ominal
> - **O**rdinal
> - **I**nterval
> - **R**atio

Each scale is **more powerful** than the previous one. Ratio > Interval > Ordinal > Nominal.

---

### 🔵 1. Nominal Scale

- **Lowest level** of measurement
- Numbers are used only as **labels/names** — no mathematical meaning
- Just **categorizes** or **classifies** data
- No ranking, no order, no distance, no true zero

| Property | Nominal |
|----------|---------|
| Classification | ✅ Yes |
| Order/Ranking | ❌ No |
| Equal Intervals | ❌ No |
| True Zero | ❌ No |

**Examples:**
- Gender: Male = 1, Female = 2 (2 is NOT greater than 1 here!)
- Religion, Nationality, Brand Preference (Pepsi=1, Coke=2)
- PIN codes, Jersey numbers

**Statistics Allowed:** Mode, Frequency, Chi-Square test
**Cannot use:** Mean, Standard Deviation

---

### 🟡 2. Ordinal Scale

- **Ranks** objects in order (1st, 2nd, 3rd...)
- Shows **relative position** but NOT the **distance** between positions
- We know A > B > C but NOT by how much

| Property | Ordinal |
|----------|---------|
| Classification | ✅ Yes |
| Order/Ranking | ✅ Yes |
| Equal Intervals | ❌ No |
| True Zero | ❌ No |

**Examples:**
- Customer satisfaction: Poor < Average < Good < Excellent
- Ranking of preferred cola brands (Rank 1, 2, 3)
- Socioeconomic class: Lower, Middle, Upper
- Army Rank: Soldier < Lieutenant < Captain

**Statistics Allowed:** Median, Mode, Percentile, Rank correlation
**Cannot use:** Mean, Standard Deviation

---

### 🟢 3. Interval Scale

- Has **equal intervals** between points
- Shows both **order AND distance**
- **No true zero** — zero is arbitrary (doesn't mean "absence of")
- Can do addition and subtraction, NOT multiplication/division

| Property | Interval |
|----------|---------|
| Classification | ✅ Yes |
| Order/Ranking | ✅ Yes |
| Equal Intervals | ✅ Yes |
| True Zero | ❌ No |

**Examples:**
- **Temperature** (Celsius/Fahrenheit): 0°C doesn't mean "no temperature"
- **Likert Scale** (1 to 5 or 1 to 7)
- **Semantic Differential Scale**
- IQ scores, Calendar years (Year 0 is arbitrary)

**Key Point:** 40°C is NOT "twice as hot" as 20°C (because zero is arbitrary)

**Statistics Allowed:** Mean, Standard Deviation, Correlation, t-test, ANOVA
**Cannot use:** Ratios (twice as much, half as much)

---

### 🔴 4. Ratio Scale

- **Highest level** of measurement
- Has **true zero** (zero means complete absence of the attribute)
- All mathematical operations allowed (+, -, ×, ÷)
- Can make ratio comparisons: "A earns twice as much as B"

| Property | Ratio |
|----------|-------|
| Classification | ✅ Yes |
| Order/Ranking | ✅ Yes |
| Equal Intervals | ✅ Yes |
| True Zero | ✅ Yes |

**Examples:**
- **Weight, Height, Age, Income**
- Number of products purchased
- Market share (%)
- Sales revenue (₹0 means no sales at all)
- **Kelvin temperature scale** (0K = absolute zero)

**Statistics Allowed:** All — Mean, SD, Geometric Mean, Coefficient of Variation, all ratios

---

### 📊 Master Comparison Table — 4 Scales

| Feature | Nominal | Ordinal | Interval | Ratio |
|---------|---------|---------|----------|-------|
| Classify | ✅ | ✅ | ✅ | ✅ |
| Rank/Order | ❌ | ✅ | ✅ | ✅ |
| Equal Distance | ❌ | ❌ | ✅ | ✅ |
| True Zero | ❌ | ❌ | ❌ | ✅ |
| Best Stat | Mode | Median | Mean | All |
| Example | Gender | Ranking | Likert Scale | Income |

> [!TIP]
> **Memory trick for scale properties:** Each new scale ADDS one property.
> Nominal → +Order = Ordinal → +Equal Intervals = Interval → +True Zero = Ratio

## 3. 🔧 SCALING TECHNIQUES

Scaling techniques are divided into **two broad categories:**

```
Scaling Techniques
├── A. Comparative Scales
│   ├── 1. Paired Comparison
│   ├── 2. Rank Order Scale
│   └── 3. Constant Sum Scale
└── B. Non-Comparative Scales (Monadic)
    ├── 1. Continuous Rating Scale
    └── 2. Itemized Rating Scales
        ├── a. Likert Scale
        ├── b. Semantic Differential Scale
        └── c. Stapel Scale
```

> [!TIP]
> **Mnemonic: "CAN vs LiSS"**
> - **Comparative** = **C**onstant sum, P**A**ired, Ra**N**k order
> - **Non-Comparative** = **Li**kert, **S**emantic Differential, **S**tapel

---

## A. COMPARATIVE SCALING TECHNIQUES

- Respondents compare objects **directly against each other**
- Results are **relative** (A is better THAN B, not how much better)
- Good for small number of objects

---

### 1. 🔄 Paired Comparison Scale

- Respondent compares **two objects at a time** and picks the preferred one
- If n = number of brands, then number of pairs = **n(n-1)/2**

**Example:**
> "Which do you prefer?"
> - Pepsi vs Coke → Pepsi ✅
> - Pepsi vs Thums Up → Thums Up ✅
> - Coke vs Thums Up → Coke ✅

**Advantages:**
- Simple to use
- Clear and unambiguous

**Disadvantages:**
- If many brands: 10 brands = 45 pairs! Too many
- Only gives relative preference, not intensity
- Violates transitivity sometimes (A>B, B>C, but C>A!)

---

### 2. 🥇 Rank Order Scale

- Respondents **rank all objects simultaneously** from most to least preferred
- Gives ordinal data

**Example:**
> Rank these cola brands from 1 (most preferred) to 4 (least preferred):
> - Pepsi: __
> - Coke: __
> - Thums Up: __
> - Sprite: __

**Advantages:**
- Closely resembles real-life decision making
- Easy to administer, less tiring than paired comparison

**Disadvantages:**
- Only ordinal — doesn't show how much better
- Can't determine if respondent likes all or hates all

---

### 3. 💰 Constant Sum Scale

- Respondents **allocate a fixed number of points** (usually 100) among objects
- Points show **relative importance/preference**

**Example:**
> Divide 100 points among these attributes of a smartphone:
> - Camera: 35
> - Battery: 25
> - Price: 20
> - Brand: 20
> - **Total: 100**

**Advantages:**
- Shows relative importance clearly
- Ratio-like data

**Disadvantages:**
- Difficult for respondents (must add to 100)
- Hard if many items

---

## B. NON-COMPARATIVE SCALING TECHNIQUES (Monadic Scales)

- Respondent rates each object **independently** (not compared to others)
- Most commonly used in marketing research
- Results are **absolute** (not relative)

---

### 1. 📊 Continuous Rating Scale (Graphic Rating Scale)

- Respondent marks a point on a **continuous line**
- No fixed categories

**Example:**
```
How do you rate the taste of this cola?
Extremely Bad |_________________________| Extremely Good
                         ↑
                    Mark here
```

**Advantages:** Very sensitive, captures nuances
**Disadvantages:** Hard to score/analyze; inconsistent

---

### 2a. ❤️ LIKERT SCALE *(Most Important!)*

> Named after **Rensis Likert** (1932)

- Also called **Summated Rating Scale**
- Respondents indicate level of **agreement or disagreement**
- Typically **5-point or 7-point** scale
- Produces **Interval** level data

**Standard Format:**
| 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|
| Strongly Disagree | Disagree | Neutral | Agree | Strongly Agree |

**Example:**
> "This product offers good value for money."
> ○ Strongly Disagree  ○ Disagree  ○ Neutral  ○ Agree  ○ Strongly Agree

**How to Score:**
- Sum up all item scores for each respondent
- Higher total = more positive attitude

**Advantages:**
- Easy to construct and administer
- Versatile — can measure many attitudes
- Respondents understand it easily

**Disadvantages:**
- Doesn't capture intensity beyond agree/disagree
- **Central tendency bias** — people pick middle (3)
- **Acquiescence bias** — tendency to agree with everything
- Doesn't show HOW FAR apart the positions are

---

### 2b. 🌊 SEMANTIC DIFFERENTIAL SCALE *(Second Most Important!)*

> Developed by **Charles Osgood** (1957)

- Bipolar adjective pairs on **opposite ends** of a scale (usually 7-point)
- Measures the **image/attitude** toward a concept or brand
- Excellent for **brand image studies**
- Also produces **Interval** level data

**Example:**
> Rate your perception of Zomato on these dimensions:
```
Fast     |_7_|_6_|_5_|_4_|_3_|_2_|_1_|     Slow
Good     |_7_|_6_|_5_|_4_|_3_|_2_|_1_|     Bad
Cheap    |_7_|_6_|_5_|_4_|_3_|_2_|_1_|     Expensive
Reliable |_7_|_6_|_5_|_4_|_3_|_2_|_1_|     Unreliable
```

**Osgood's 3 Dimensions (EPA):**
| Dimension | Example Bipolar Pairs |
|-----------|----------------------|
| **E**valuation | Good-Bad, Pleasant-Unpleasant |
| **P**otency | Strong-Weak, Heavy-Light |
| **A**ctivity | Fast-Slow, Active-Passive |

> **Mnemonic for EPA: "Every Product has Activity"**

**Advantages:**
- Captures brand image powerfully
- Can compare multiple brands on same chart (snake diagram)
- Versatile across cultures

**Disadvantages:**
- Choosing right bipolar pairs is subjective
- Some concepts hard to express as opposites
- Halo effect — one good attribute makes all seem good

---

### 2c. 📍 STAPEL SCALE

- Modified version of Semantic Differential
- **Unipolar** — only ONE adjective, no opposite
- Scale ranges from **-5 to +5** (11 points, no zero label)
- No neutral midpoint label

**Example:**
> Rate "Swiggy" on this attribute:
```
         +5
         +4
         +3
FAST     +2
         +1
         -1
         -2
         -3
         -4
         -5
```
> Positive numbers = adjective describes brand well; Negative = doesn't describe well

**Advantages:**
- Easier to construct (only one adjective needed)
- Can be administered by phone easily
- No need to pretest for bipolarity

**Disadvantages:**
- Less commonly used; unusual format confuses respondents
- Results harder to interpret than Semantic Differential

---

### 🎯 Thurstone Scale (Equal Appearing Interval)

- Developed by **L.L. Thurstone** (1929) — oldest attitude scale
- Expert judges sort 100+ statements into 11 piles from most to least favorable
- Average judge rating becomes the **scale value** of each statement
- Final scale has ~20 statements with known scale values
- Respondents check items they agree with

**Advantages:** Scientifically rigorous
**Disadvantages:** Very time-consuming to construct; judges may be biased
**Use:** Academic research, not common in commercial MR

---

### 📊 Comparison: Likert vs Semantic Differential vs Stapel

| Feature | Likert | Semantic Differential | Stapel |
|---------|--------|----------------------|--------|
| Format | Agreement statements | Bipolar adjective pairs | Single adjective |
| Scale | 5 or 7 point | 7 point | -5 to +5 (10 pt) |
| Data Type | Interval | Interval | Interval |
| Best For | Attitude measurement | Brand image | Quick attribute rating |
| Ease of Use | Easy | Moderate | Easy |
| Developed by | Likert (1932) | Osgood (1957) | Stapel (1950) |

## 4. ✅ RELIABILITY AND VALIDITY

These two concepts determine if your scale is **trustworthy and accurate**.

> **Analogy:** A bathroom weighing scale that always shows you 5 kg less is:
> - **Reliable** ✅ (consistently wrong — same error every time)
> - **NOT Valid** ❌ (doesn't measure true weight)
>
> A scale that shows random different weights each time is:
> - **NOT Reliable** ❌
> - **NOT Valid** ❌

---

### 🔁 RELIABILITY

> **Definition:** Reliability is the degree to which a measure is **free from random error** and therefore gives **consistent results** across repeated measurements.

- A reliable scale gives the **same result** when measuring the same thing repeatedly
- Reliability is a **necessary but NOT sufficient** condition for validity
- High Reliability → Low Random Error

### Types of Reliability:

> [!TIP]
> **Mnemonic: "TEST-IH"**
> - **T**est-Retest
> - **E**quivalent Forms (Alternative Forms)
> - **S**plit-Half
> - **T**hree above are main types
> - **I**nternal Consistency (Cronbach's Alpha)
> - **H**inter: Reliability = Consistency

#### 1. Test-Retest Reliability
- Same scale administered to **same respondents at two different times**
- If results are similar → Reliable
- **Problem:** Respondents may remember answers, learning effect

#### 2. Alternative/Equivalent Forms Reliability
- Two **different but equivalent** versions of the scale given to same group
- Correlation between two versions = reliability
- **Problem:** Hard to create truly equivalent forms

#### 3. Internal Consistency Reliability
- Tests if all items in a scale measure the **same construct**
- Most common method: **Cronbach's Alpha (α)**
  - α ranges from 0 to 1
  - α ≥ 0.7 = Acceptable reliability
  - α ≥ 0.8 = Good reliability
  - α ≥ 0.9 = Excellent reliability
- **Split-Half Method:** Divide scale into two halves, correlate scores

---

### 🎯 VALIDITY

> **Definition:** Validity is the degree to which a scale **actually measures what it is supposed to measure**.

- A valid scale is accurate — it hits the target
- Validity is harder to establish than reliability
- A scale CAN be reliable but invalid (consistent but measuring wrong thing)
- If a scale is Valid → it MUST also be Reliable

### Types of Validity:

> [!TIP]
> **Mnemonic: "CCC" — Content, Criterion, Construct**
> 
> *"Content ke Criterion se Construct banta hai!"*

#### 1. 📚 Content Validity (Face Validity)
- Does the scale **look like** it measures what it should?
- Judged by **experts/panel** reviewing the items
- Subjective — no statistical test
- **Example:** A test for "marketing knowledge" should have questions on all marketing topics, not just one area

#### 2. 📏 Criterion Validity (Predictive/Concurrent)
- Does the scale correlate with an **external criterion** (another established measure)?
- Two subtypes:
  - **Concurrent Validity:** Correlation with criterion measured at the **same time**
    - Example: New attitude scale vs old established attitude scale
  - **Predictive Validity:** Scale score predicts a **future criterion**
    - Example: MBA entrance test score predicts MBA performance

#### 3. 🏗️ Construct Validity
- Does the scale measure the **theoretical construct** it's supposed to?
- Most sophisticated type of validity
- Two subtypes:
  - **Convergent Validity:** Scale correlates **highly** with other scales measuring the **same** construct (they should agree)
  - **Discriminant Validity:** Scale correlates **poorly** with scales measuring **different** constructs (they should differ)

---

### 🎭 Reliability vs Validity — The Target Analogy

```
Reliable but NOT Valid    Valid and Reliable    Neither
(Consistently off-target) (Consistently on-target) (Randomly off)
        ○ ○                      ●                    ○
      ○   ○                    ●●●                  ○   ○
        ○ ○                      ●                    ○
```

### Reliability vs Validity Comparison:

| Feature | Reliability | Validity |
|---------|-------------|---------|
| Meaning | Consistency | Accuracy |
| Measures | Freedom from random error | Measures what it should |
| Test | Cronbach's Alpha, Test-Retest | Expert judgment, Correlation |
| Relationship | Necessary for validity | Sufficient (implies reliability) |
| Can exist without other? | Yes (reliable but not valid) | No (valid = must be reliable) |

## ⚡ QUICK REVISION FLASHCARDS

**Q: What is the key difference between Interval and Ratio scale?**
A: Ratio has TRUE ZERO (e.g., Income ₹0 = no income). Interval has arbitrary zero (e.g., 0°C doesn't mean no temperature)

**Q: Which scale is used for brand image studies?**
A: Semantic Differential Scale (bipolar adjective pairs)

**Q: Likert scale produces what type of data?**
A: Interval scale data

**Q: What does Cronbach's Alpha measure?**
A: Internal Consistency Reliability. α ≥ 0.7 is acceptable.

**Q: Can a scale be reliable but not valid?**
A: YES! A scale can consistently give wrong measurements (reliable but invalid). But if valid → must be reliable.

**Q: What are the 3 types of validity?**
A: Content (face), Criterion (concurrent/predictive), Construct (convergent/discriminant)

**Q: Paired comparison of 5 brands = how many pairs?**
A: n(n-1)/2 = 5×4/2 = **10 pairs**

**Q: Thurstone scale was developed by whom and when?**
A: L.L. Thurstone, 1929 (oldest attitude scale)

**Q: Stapel scale range?**
A: -5 to +5 (no zero label, no opposite adjective)

**Q: Nominal scale — which statistic is allowed?**
A: Mode and Chi-Square. Mean/SD NOT allowed.
